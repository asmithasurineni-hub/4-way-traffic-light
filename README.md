# 4-Way Traffic Light Controller FSM

## Project Overview
This project implements a 4-Way Traffic Light Controller using Verilog HDL and Finite State Machine (FSM) concepts.

The controller manages traffic signals at a four-way intersection by sequencing Red, Yellow, and Green lights for North-South and East-West traffic directions. The design is implemented as a synchronous FSM and verified through simulation.

## Features
- FSM-based traffic signal control
- Verilog HDL implementation
- Synchronous design
- Testbench verification
- Simulation waveform analysis
